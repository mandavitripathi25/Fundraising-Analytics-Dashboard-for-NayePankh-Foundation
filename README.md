# Fundraising-Analytics-Dashboard-for-NayePankh-Foundation
Objective  Analyze donation patterns, donor behavior, and fundraising performance using Data Analytics.
Project Workflow
Step 1: Create Sample Dataset
Step 2: Data Exploration
Step 3: Monthly Donation Trend
Step 4: Donor Growth Analysis
Step 5: Volunteer Engagement Analysis
Step 6: KPI Dashboard
Step 7: Automated Report Generation
**Findings**
1)Donations increased from ₹50,000 in January to ₹1,20,000 in June.
2)Donor count steadily increased throughout the period.
3)Volunteer participation showed positive growth.
4)June was the best-performing month for fundraising.
5)Growth suggests successful awareness and engagement efforts
